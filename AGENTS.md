## Project Summary
A professional web application for NoctraTech, featuring a hero section and a contact form with Gmail integration. The project is built with Next.js and optimized for performance.

## Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS, Framer Motion
- Email: Nodemailer (via Gmail SMTP)
- Runtime: Bun

## Architecture
- `src/app`: Page routes and API endpoints
- `src/components`: UI components and sections
- `src/lib`: Utility functions and hooks

## User Preferences
- Gmail as primary email service (`technoctra@gmail.com`)

## Project Guidelines
- No comments unless requested
- Use functional components
- Follow Next.js App Router conventions
- Optimize images using `next/image`

## Common Patterns
- Framer Motion for animations
- Server Actions/API routes for form submissions
